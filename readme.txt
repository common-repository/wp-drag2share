== Description ==

WP-Drag2Share is a simple tool to share a post easily by drag and drop, this plugin supports social networks like Twitter, Delicious and Facebook.

This plugin is based on the script developed by Dan Wellman, achieving a better interface with

== Installation ==

This section describes how to install the plugin and get it working.

1. UnZip and Upload 'wp-drag2share.zip' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the image which you want to have the "Drag and Share" effect between &lt;div class=&quot;ui-draggable&quot;&gt; and &lt;/div&gt;
